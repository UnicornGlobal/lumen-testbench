## 5.0.0

Released: 2020-02-29

### Changes

* Update support for Laravel Framework v7.
* Increase minimum PHP version to 7.2.5 and above (tested with 7.2, 7.3 and 7.4).
* Configuration changes:
    - Changed `MAIL_DRIVER` to `MAIL_MAILER` environment variable.
